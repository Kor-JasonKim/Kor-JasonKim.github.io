01
AI 기반 방 청결도 분석 시스템
2026.03팀 프로젝트 (4인) AI · Computer Vision · Web AI 비전 우수상

사람마다 다른 주관적인 '깨끗함'을 데이터로 정량화하는 서비스입니다. 사진 한 장으로 방·책상의 청결도 점수를 매기고, 어디가 왜 지저분한지를 히트맵과 자연어로 함께 제시합니다. 분류·객체 탐지·이미지-텍스트 세 모델을 결합하고, 사용자 평가와 비교하는 커뮤니티까지 구현했습니다.

개요

Overview
문제
1인 가구 증가 속, 청결의 객관적 기준 부재. ECCS 척도와 선행 논문을 조사해 출발점으로 삼음.
목표
정리가 필요한 구역을 시각화하고, 누구나 납득할 객관적 청결 지수를 수립.
결과
방·책상 모델 모두 F1 0.95+ 달성. 점수·근거·커뮤니티를 갖춘 웹 서비스로 완성.

담당 업무

My Role
  • CNN(Keras·ResNet50) 청결도 판정 모델 설계 및 구축 — 방·책상 Clean/Dirty 이진 분류
  • train / val / test 분리, 하이퍼파라미터 튜닝, Early Stopping 적용으로 과적합 제어
  • Flask 기반 추론 서버 프로토타입 설계 — 모델을 웹에서 호출하는 파이프라인 구성
  • 시연 영상 기획 및 편집

핵심 구현

How it works
01종합 청결도 점수

CNN 분류 확률과 YOLO 객체 분포를 가중합해 0~100 종합 지수를 산출. 단일 모델의 편향을 보완.

total = w_k·k + w_y·y  (시연 가중치 CNN 0.8 / YOLO 0.2)

02설명 가능한 판정
  • Grad-CAM 히트맵으로 모델이 주목한 영역 시각화
  • CLIP으로 "무엇이 왜 지저분한지" 자연어 설명 생성
03실시간 모니터링

Raspberry Pi 홈캠이 30초 간격으로 촬영하고, 5회 연속 Dirty 판정 시 부저로 알림.

04유저 커뮤니티

사용자가 서로의 방 사진에 점수를 매기고 AI 점수와 비교. 향후 재학습용 데이터로 활용 가능.

트러블슈팅

Troubleshooting
문제
어두운 조명의 깨끗한 방을 더러움으로 오분류 — 모델이 밝기에 과도하게 반응.
해결
Grayscale 변환 + 데이터 증강으로 색이 아닌 형태·배치에 집중하도록 학습 유도.
문제
판정 확률이 99%·98% 같은 극값에 치우치는 과적합 경향.
해결
Early Stopping 적용으로 과적합 위험을 낮추고 일반화 성능 확보.
문제
일부 이미지에서 사람과 모델의 인식 차이(데이터셋 편향) 발생.
해결
Grad-CAM 결과로 편향 영역을 파악해 해당 유형 데이터를 추가 수집.
문제

해결

STM32 스마트 알약 자동 디스펜서 · Pill-O-Clock
2026.04팀 프로젝트 (4인) Embedded · Firmware · App

복약 시간과 요일을 앱에서 설정하면, 정해진 시각에 해당 요일의 약이 자동으로 배출되는 디스펜서입니다. 약통 회전·배출·복용 확인까지 하나의 상태 머신으로 제어하고, 펌웨어는 HAL 없이 레지스터를 직접 다루는 순수 C로 작성했습니다. 기구 설계와 3D 프린팅까지 직접 진행했습니다.

개요

Overview
목표
블루투스 기반 자동 알약 배출 시스템 구현 및 복약 관리 자동화.
내용
STM32 마이크로컨트롤러로 스마트 디스펜서를 제작하고 센서·모터·앱을 통합.
결과
시간·요일별 자동 배출 시스템 완성 및 연속 동작 안정성 검증 완료.

담당 업무

My Role
  • HAL 없이 레지스터를 직접 제어하는 펌웨어 작성 (Bare-metal C)
  • 시간 기반 동작을 제어하는 상태 머신 설계 — IDLE → FORWARD → WAIT → BACKWARD → FINISHED
  • 타이머·인터럽트(NVIC) 스케줄링, 스텝/서보 모터 구동 제어
  • HC-05 UART 양방향 통신 및 앱 연동 데이터 처리
  • MIT App Inventor 기반 제어 앱 시제품 제작

핵심 구현

How it works
01정밀 위치 제어 (스텝모터)

요일별 약통을 정확한 각도로 돌리기 위해 펄스 수를 직접 계산해 적용.

02배출 메커니즘 (서보)
  • PWM으로 0°~45° 배출구 개폐 제어
  • 상부 저장공간에서 한 칸씩 순차 보충
03시간 기반 자동 동작

내부 타이머·RTC로 시각을 추적하다 설정 시각 도달 시 자동 배출. LCD·부저로 알림.

04센서 안전 제어
  • 초음파 거리 ≤ 1cm 시 정지 + 부저 ON
  • 복용 확인 후 ≥ 15cm 복귀 / PSC·ARR 속도 튜닝
# 요일별 회전  360° ÷ 7 ≈ 51.4° / day
# 필요 스텝  32 step × 64 (내부 기어비) × 7.8 (약통 기어비) ÷ 7
  = ≈ 2,282 steps

음성 인식 기반 장애물 회피 자율주행
2025.03 – 07경희대 졸업 종합설계 ROS2 · Deep Learning 종합설계 우수상

LiDAR·카메라로는 볼 수 없는 코너 너머의 위험을 '소리'로 먼저 읽는 자율주행 접근입니다. 직접 설계한 경량 음원 인식 모델로 장애물의 방향을 추정해, 충돌이 일어나기 전에 미리 우회 경로를 계획합니다.

개요

Overview
문제
코너·사각지대처럼 LiDAR·카메라 시야가 막히는 구간에서는 장애물을 미리 인지하기 어려움.
접근
소리의 방향·종류를 추정해, 보이지 않는 위험을 충돌 전에 '소리'로 먼저 감지하는 음향 단서 결합.
결과
경량 SELD 모델 MMT-Net 설계 + 코너 트리거존 기반 사전 회피 주행 구현. 졸업 종합설계 우수상 수상.

담당 업무

My Role
  • 코너 탐지 알고리즘 및 ROS 노드 구현 — 시야가 막히는 지점을 판별
  • 소리 감지 시 우회를 트리거하는 트리거존 로직 설계
  • 주행 맵 전처리 파이프라인 구축
  • 경량 음원 인식 모델 MMT-Net (Transformer + TCN) 설계

핵심 구현

How it works
01음원 방향 추정 (SELD)

마이크 입력에서 소리의 발생 방향과 종류를 추정해 장애물의 위치를 예측.

02코너 트리거존

시야가 막히는 코너에 트리거존을 설정, 소리가 감지되면 사전 회피를 발동.

03사전 경로 재계획

충돌 전에 Nav2 경로를 다시 계산해 넓게 우회하는 주행을 수행.

04모델 경량화

임베디드 배포를 염두에 두고 Conformer 대비 약 6× 가벼운 구조로 설계.

# MMT-Net  Multi-scale Transformer + TCN
# 입력 → 출력  멀티채널 오디오 → 음원 방향(DOA) · 종류 동시 추정 (SELD)
# 경량화  Conformer 比 ~6× ↓ · 2.4 M params · 추론 1.086 ms

붕괴·화재·유독가스로 사람이 진입하기 어려운 재난 환경을 로봇이 스스로 탐사하는 시스템입니다. 조종 없이 미지의 지형을 주행하며 실시간으로 지도를 작성하고 조난자 위치를 파악하는 자율 탐사를 담당하고, 중앙 관제가 다중 로봇을 조율하는 구조로 설계했습니다.

개요

Overview
문제
붕괴·화재·유독가스로 진입이 위험하고, 사전 지도가 없는 미지 공간에서의 탐색이 필요.
목표
수동 조종 없이 자율 탐사·실시간 지도 작성·조난자 위치 파악, 중앙 관제로 다중 로봇 운용.
결과
프론티어 탐사 + slam_toolbox + Nav2 통합, 도메인 격리 다중 로봇 관제를 실환경에서 검증.

  • slam_toolbox 실시간 SLAM 맵핑 — 주행과 동시 지도 작성, 탐사 완료 시 자동 복귀·지도 저장

핵심 구현

How it works
01프론티어 자율 탐사

가장 가까운 미탐색 경계로 목표를 자동 생성하고, Clustering으로 유효 프론티어만 선별해 탐사.

02실시간 SLAM 맵핑

slam_toolbox로 주행과 동시에 지도를 작성, 탐사 범위·위치를 지속 갱신.

03Nav2 경로 계획

노드 단위 목표 전송으로 호출 빈도를 낮춰 네트워크 병목을 해소하며 주행.
